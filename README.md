# Taya RPA Challenge
### Description
RPA that login in Taya webpage and simulates real estate credit using Google Chrome Browser.
### Preview
![Captura de tela 2023-11-27 232103](https://github.com/issitarual/taya-rpa/assets/81389078/dd0c86d6-b43b-44b9-bb2b-79052bf1fbd2)
### Requirements
- [x]  Go to Taya webpage
- [x]  Login using an e-mail and password
- [x]  Access the simulator
- [x]  Fill user data fields
- [x]  Fill property data fields
- [x]  Calculate the real estate credit
